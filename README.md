# Sweet Box Website

Editorial website design for sweetbox.co.il — American Bakeshop in Tel Aviv.

## Pages
- `index.html` — Homepage
- `product-bestseller.html` — Best Seller Box product page

## Tech
- Plain HTML/CSS (no build needed)
- Hebrew RTL
- Fonts: Italiana, Cormorant Garamond, Frank Ruhl Libre
- Brand colors: powder blue #A1D9FA, warm brown #554B41, cream #F4EFE6

## Deployment
GitHub Pages auto-deploys from `main` branch root.
